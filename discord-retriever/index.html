Qu1ck5h0t's very functional discord message log retriever!

Don't mind the 2005 UI, I'm not a front end dev!

This will pack all messages sent within a channel between the given start date and end date into a zip file to download.

Your Discord token remains between your client and Discord's API, feel free to inspect the source code (this site runs on Github Pages).
